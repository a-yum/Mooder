# Mooder
**Track your mood with Mooder!**



<h2>Features</h2>
<li>CRUD Operations for Mood Entries</li>
<li>Filter Mood Entries by Mood or Date</li>
<li>Data Visualization</li>

<h2>Technologies</h2>
<li>.Net</li>
<li>Sqlite</li>
<li>Bootstrap</li>
